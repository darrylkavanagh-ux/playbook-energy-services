---

## ⚠️ **IF THE IMPORT BUTTON IS GREYED OUT**

**What it means:**
- The repository is already connected to an existing Vercel project
- You need to delete that old project first

**How to fix it:**

### Quick fix (2 minutes):
1. **Tap this link:** 👉 https://vercel.com/dashboard
2. **Look for** any project named:
   - "playbook-energy-services"
   - "playbook-energy"
   - or anything with "playbook" in the name
3. **Tap** that project
4. **Tap** Settings (at the bottom)
5. **Scroll** to the very bottom
6. **Tap** the red "Delete Project" button
7. **Type** the project name to confirm
8. **Tap** Delete
9. **Go back** to: 👉 https://vercel.com/new
10. The **Import** button should now work!
