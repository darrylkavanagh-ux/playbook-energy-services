## 🆘 TROUBLESHOOTING

### Problem at Step 14: Can't edit Root Directory

**The pencil icon doesn't work**

**Solution:**
1. Close the tab
2. Open Chrome or Edge (NOT Safari)
3. Go back to Step 6 (Incognito mode)
4. Try again

---

### Problem at Step 21: "Project name already exists"

**Error message when clicking Deploy**

**Solution:**
1. Go back to Step 12
2. Change project name to: `foxlite-april-9-final`
3. Continue with Step 13-21 again

---

### Problem at Step 22: Build fails with "pnpm" error

**Logs show error about "pnpm-lock.yaml"**

**Solution:**
1. The Root Directory is wrong
2. Click "Cancel Build" button
3. Go back to Vercel dashboard
4. Find the project and click it
5. Go to Settings → General
6. Find "Root Directory"
7. Change it to: `website`
8. Go to Deployments → Click "..." menu → Redeploy

---

### Problem at Step 22: Build takes more than 10 minutes

**Still building after 10 minutes**

**Solution:**
1. Something is wrong
2. Click anywhere outside the logs box
3. Look for "Cancel" button
4. Click it
5. Take a screenshot
6. Send screenshot for help
