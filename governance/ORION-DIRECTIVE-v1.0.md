## SECTION 4: COMPLETE INVESTIGATION WORKFLOW

### THE ORION INVESTIGATION PROTOCOL

Every investigation follows this sequence automatically:

```
STAGE 1: INTAKE (ENG-03 OMNISEARCH + ENG-15 IRIS)
  ↓ Subject ingested, documents classified, initial data gathered

STAGE 2: IDENTITY & SCREENING (ENG-18 SANCTIONS + ENG-30 OSINT)
  ↓ Sanctions check, PEP check, adverse media, corporate connections

STAGE 3: CORPORATE INTELLIGENCE (ENG-19 CORPORATE + ENG-20 UBO)
  ↓ All companies mapped, UBOs traced, offshore structures unravelled

STAGE 4: FINANCIAL INTELLIGENCE (ENG-28 FORENSIC + ENG-17 CRYPTO)
  ↓ Bank statement analysis, crypto tracing, net worth analysis

STAGE 5: LEGAL INTELLIGENCE (ENG-27 COURT + ENG-22 PROFESSIONAL)
  ↓ Court records pulled, professional status checked, orders identified

STAGE 6: DEEP INVESTIGATION (ENG-04 CVK-1100 → ENG-05 JIGSAW → ENG-02 SENTINEL)
  ↓ Everything verified, assembled, graded NATO S/C scale

STAGE 7: PATTERN ANALYSIS (ENG-11 PRISM + ENG-12 ATLAS + ENG-29 TIMELINE)
  ↓ Network mapped, timeline reconstructed, anomalies flagged

STAGE 8: DARK INTELLIGENCE (ENG-26 DARK_WEB + ENG-03 OMNISEARCH extended)
  ↓ Dark web checked, OSINT completed, all open sources exhausted

STAGE 9: INSOLVENCY ANALYSIS (ENG-21 INSOLVENCY)
  ↓ (if applicable) IP fraud, asset stripping, clawback analysis

STAGE 10: ASSET RECOVERY PLANNING (ENG-23 ASSET_RECOVERY)
  ↓ Recovery mechanisms identified, applications drafted

STAGE 11: VERIFICATION (ENG-06 VERITECH — VeriTech-10 Protocol)
  ↓ EVERY finding verified 10 points, human sign-off gate

STAGE 12: OUTPUT GENERATION
  ├── ENG-24 PROSECUTION_BUNDLE (criminal referral)
  ├── ENG-25 SAR_GENERATOR (NCA/Garda FIU SAR)
  ├── ENG-22 COMPLAINT (regulatory complaint)
  ├── ENG-23 ASSET_RECOVERY (freezing order application)
  ├── ENG-31 MLAT (international co-operation)
  └── ENG-13 CHRONICLE (full investigation report)

STAGE 13: INTERNATIONAL CO-OPERATION (ENG-31 MLAT)
  ↓ (if required) MLA requests, Interpol notices, foreign enforcement

STAGE 14: HUMAN SIGN-OFF (VeriTech Level 10)
  ↓ Qualified professional reviews and certifies output

STAGE 15: DELIVERY
  → Prosecution bundle to: CPS / DPP / Garda / SFO / CAB
  → SAR to: NCA goAML / Garda FIU
  → Complaint to: SRA / FCA / Law Society IE / ICAI
  → Application to: High Court (freezing order / UWO / CAB)
  → MLAT to: Central Authority (DOJ / Ministry of Justice)
```
