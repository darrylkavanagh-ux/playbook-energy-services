## ⚡ 3 STEPS TO LIVE WEBSITE

### STEP 1: Deploy to Vercel (15 minutes)

1. Go to: https://vercel.com/dashboard
2. Click: **"Add New..."** → **"Project"**
3. Import: `darrylkavanagh-ux/playbook-energy-services`
4. Root Directory: `website`
5. Click: **"Deploy"**
6. Add domains: `foxliteforensics.com` + `www.foxliteforensics.com`
7. ✅ **LIVE at https://foxliteforensics.com**

### STEP 2: Configure Email (30 minutes)

1. Go to: https://dash.cloudflare.com
2. Add site: `foxliteforensics.com` (Free plan)
3. Update nameservers at your domain registrar
4. Enable: **Email → Email Routing**
5. Create 4 addresses:
   - info@foxliteforensics.com
   - audit@foxliteforensics.com
   - support@foxliteforensics.com
   - accounts@foxliteforensics.com
6. ✅ **EMAIL WORKING**

### STEP 3: Test & Verify (15 minutes)

1. Visit: https://foxliteforensics.com
2. Check all pages load
3. Verify HTTPS/SSL
4. Test mobile responsive
5. Send test emails to all 4 addresses
6. ✅ **ALL WORKING**
