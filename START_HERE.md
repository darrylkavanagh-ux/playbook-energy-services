## 🆘 IF YOU HAVE ISSUES

**Problem: Can't see dropdowns or edit buttons**
- Solution: Use Chrome Incognito mode (Ctrl+Shift+N)
- Or clear browser cache (Ctrl+Shift+Delete)

**Problem: "Project name already exists"**
- Solution: Use a different name like: `foxlite-forensics-prod-2026`

**Problem: "pnpm-lock.yaml" error**
- Solution: Make sure Root Directory is set to `website`, not `./`

**Problem: Repository not showing**
- Solution: Click "Adjust GitHub App Permissions" and check the box next to `playbook-energy-services`
