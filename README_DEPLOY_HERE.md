## 🚨 **IF SOMETHING GOES WRONG**

### Repository is greyed out?
**Solution:** Delete the old Vercel project first
1. Go to: https://vercel.com/dashboard
2. Find and delete any project named "playbook-energy-services"
3. Return to: https://vercel.com/new

### Build fails with "pnpm" error?
**Reason:** Root Directory is wrong (still set to `./`)
**Solution:** Set Root Directory to `website` and redeploy

### Can't tap pencil icon?
**Solution:** Rotate to landscape mode OR zoom in OR request desktop website

### Other problems?
**Read:** SCREEN_BY_SCREEN_IPHONE.md (has troubleshooting for 7 common issues)
